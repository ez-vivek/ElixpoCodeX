## üöÄ Quick Start

Set up ElixpoCodeX locally by following these steps:


### **Clone the Repository**  
Navigate to your desired directory and run:  
```bash
git clone https://github.com/ez-vivek/ElixpoCodeX.git
```

---

### **Setup**  

1. Navigate to the server directory:  
    ```bash
    npm install
    ```

3. Start the server:  
    ```bash
    npm run dev
    ```
